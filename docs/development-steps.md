# 分步开发计划

## 推进原则

- 每次只推进一个小里程碑。
- 每个里程碑开始前，先确认目标、范围和验收标准。
- 每个里程碑完成后，更新当天开发日志。
- 用户未确认前，不进入下一里程碑。
- 不确定事项记录到对应文档，不凭空决定。

## 里程碑

### M0：文档与工作规范初始化

目标：

- 创建 `docs/`、`dev-journal/`、`CLAUDE.md`。
- 建立需求、技术、设计、执行和验收标准文件。
- 不写应用功能代码。

验收：

- M0 所需文件全部存在。
- `CLAUDE.md` 包含工作前阅读路径和日志更新规则。

### M1：技术路线确认

目标：

- 对 Electron、.NET 桌面方案做简要对比。
- 根据剪贴板、托盘、快捷键、开机自启、打包等需求确认第一版技术栈。

验收：

- `docs/technical-plan.md` 更新为已确认技术路线。
- 用户明确同意进入 M2。

### M2：项目骨架

目标：

- 创建最小可运行桌面应用。
- 只验证窗口能打开、项目能启动。

验收：

- 本地可启动应用窗口。
- 不实现业务功能。

状态：

- 已完成。
- 已创建最小 Electron + TypeScript 应用骨架。
- 已通过 smoke 测试验证窗口加载。

### M3：文字剪贴板监听

目标：

- 实现文字复制记录。
- 支持展示文字历史。
- 支持点击历史记录后重新写入剪贴板。

验收：

- 复制文字后能在界面看到记录。
- 点击记录后能重新放回剪贴板。

状态：

- 已完成。
- 当前记录保存在内存中，重启应用后会丢失。
- 已通过 smoke 测试验证文字记录、展示和点击写回剪贴板。

### M4：本地历史存储

目标：

- 增加本地保存。
- 实现重复内容合并。
- 实现按时间倒序读取。

验收：

- 重启软件后历史记录仍存在。
- 相同内容重复复制时只保留一条并更新时间。

状态：

- 已完成。
- 当前文字历史保存到 Electron `userData` 目录下的 `text-history.json`。
- 已通过 smoke 测试验证持久化、重复合并和倒序读取。

### M5：基础 UI

目标：

- 实现搜索框、历史卡片、删除、置顶。
- 保持简洁直观的淡蓝色风格。

验收：

- 可搜索文字记录。
- 可置顶记录。
- 可删除记录。

状态：

- 已完成。
- 已实现搜索框、文字历史卡片、置顶、删除和淡蓝色界面优化。
- 已通过 smoke 测试验证搜索、置顶、删除和持久化。

### M6：保留时间清理

目标：

- 支持 `1 天`、`3 天`、`5 天`。
- 置顶内容不自动清理。

验收：

- 未置顶过期内容会被清理。
- 置顶内容不会因过期被自动清理。

状态：

- 已完成。
- 已实现保留时长设置入口，并保存到 Electron `userData` 目录下的 `settings.json`。
- 已实现未置顶过期文字记录清理，置顶记录不会因过期自动删除。
- 已通过 smoke 测试验证保留时长设置、过期清理、置顶保留和持久化。

### M7：图片记录

目标：

- 支持图片复制记录。
- 支持图片缩略图展示。
- 支持点击图片记录后重新写入剪贴板。
- 单张图片限制 `10MB`。

验收：

- 复制图片后能显示缩略图。
- 点击图片记录后能重新放回剪贴板。
- 超过 `10MB` 的图片不保存。

状态：

- 已完成。
- 已扩展历史记录结构，支持文字和图片两种类型，并兼容旧文字历史数据。
- 已实现图片复制记录、图片缩略图展示、点击图片记录后重新写入剪贴板。
- 已实现单张图片 `10MB` 保存上限，超过上限的图片不保存。
- 已通过 smoke 测试验证图片历史、缩略图、写回剪贴板、持久化和大小限制。

### M8：托盘、快捷键、开机自启

目标：

- 支持系统托盘常驻。
- 支持 `Ctrl+Shift+V` 呼出历史面板。
- 支持开机自启。

验收：

- 关闭主窗口后仍在后台运行。
- 托盘可打开主窗口。
- 快捷键可打开历史面板。
- 开机自启设置可生效。

状态：

- 已完成。
- 已实现系统托盘常驻和托盘菜单打开/退出。
- 已实现关闭主窗口后隐藏到后台继续运行。
- 已实现 `Ctrl+Shift+V` 全局快捷键呼出历史面板。
- 已实现开机自启设置入口，并通过 Electron 登录项设置应用。
- 已通过 smoke 测试验证托盘、快捷键、关闭隐藏、窗口恢复和开机自启设置。

### M9：打包与验收

目标：

- 完成 Windows 可运行产物。
- 按测试验收文档逐项检查。

验收：

- 生成可运行软件包或可执行文件。
- 核心功能通过验收。

状态：

- 已完成。
- 已新增 `scripts/package-win.js` 本地 Windows unpacked 打包脚本。
- 已新增 `npm.cmd run package:win` 打包命令。
- 已生成 Windows 可运行产物：`release/历史剪贴板-win32-x64/历史剪贴板.exe`。
- 已通过构建、M3-M8 smoke 回归和产物启动验证。
- 当前产物为未安装版文件夹，不包含安装器、代码签名或自动更新。

### M10：主窗体验优化与常驻悬浮按钮

目标：
- 优化主窗口搜索、空状态、状态反馈、按钮可读性和基础响应式布局。
- 新增默认开启的常驻悬浮按钮。
- 支持点击悬浮按钮、托盘入口或 `Ctrl+Shift+V` 打开轻量悬浮历史面板。
- 悬浮面板第一版只支持查看最近记录和点击复制回剪贴板。

验收：
- `npm.cmd run build` 通过。
- M3-M8 smoke 回归通过。
- M10 smoke 输出 `M10_SMOKE_TEST_FLOATING_WINDOW_OK`。

状态：
- 已完成。
- 已新增独立悬浮窗页面、悬浮按钮显示设置、托盘菜单入口和悬浮窗 smoke 验收。
