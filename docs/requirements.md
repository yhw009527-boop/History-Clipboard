# 产品需求文档

## 产品目标

开发一个运行在 Windows 电脑上的历史剪贴板软件，用于记录近期复制过的文字和图片，方便用户查找历史内容并再次粘贴。

## 已确认需求

- 第一版优先支持 Windows。
- 软件开机自启并常驻后台。
- 关闭主窗口后，软件仍继续记录剪贴板内容。
- 用户可通过系统托盘图标打开历史面板。
- 用户可通过全局快捷键 `Ctrl+Shift+V` 打开历史面板。
- 点击历史记录后，将对应内容重新放回系统剪贴板，用户再手动按 `Ctrl+V` 粘贴。

## 剪贴板记录

- 支持记录文字。
- 支持记录图片。
- 图片单张大小上限为 `10MB`。
- 超过大小上限的图片不保存。
- 相同内容只保留一条。
- 用户再次复制相同内容时，更新原记录时间并移动到最新位置。

## 历史列表

- 打开软件后，以时间降序展示复制记录。
- 每条记录使用卡片形式展示。
- 文字卡片展示内容预览。
- 图片卡片展示缩略图。

## 用户操作

- 搜索文字历史记录。
- 将历史记录置顶。
- 删除单条历史记录。
- 点击记录后重新写入系统剪贴板。

## 保留时间

- 支持 `1 天`、`3 天`、`5 天`。
- 未置顶内容超过保留时间后自动清理。
- 置顶内容不受保留时间限制，除非用户手动删除。

## 暂不包含

- 云同步。
- 账号系统。
- 多设备同步。
- 本地数据加密。
- 敏感内容自动过滤。
- 图片文字识别搜索。
- 自动粘贴到当前输入位置。

## 待确认事项

- 第一版技术栈：已确认采用 Electron。
- 打包形式尚未确定。
- 数据存储格式尚未确定。

## M10 已确认升级需求

- 优先优化主窗口交互细节，包括空状态说明、状态反馈、按钮可读性和小屏布局。
- 新增默认开启的常驻悬浮按钮，位置贴近屏幕边缘并保持置顶。
- 点击悬浮按钮或使用 `Ctrl+Shift+V` 打开轻量悬浮历史面板。
- 托盘菜单保留打开主窗口入口，并新增打开悬浮面板、显示/隐藏悬浮按钮入口。
- 悬浮面板第一版只支持查看最近记录和点击复制回系统剪贴板，不包含搜索、置顶、删除。
- 用户隐藏悬浮按钮后，将显示状态保存到本地设置文件。
